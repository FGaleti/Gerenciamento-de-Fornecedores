# API de Gerenciamento de Fornecedores

Aplicação **Spring Boot** voltada para o cadastro e gerenciamento de fornecedores. Este projeto tem como propósito praticar o mapeamento JPA utilizando atributos de negócio específicos (como CNPJ e Nome Fantasia) alterando os nomes das colunas no banco de dados.

## 🚀 Tecnologias Utilizadas
* Java
* Spring Boot
* Spring Data JPA
* Banco de Dados Relacional

## 📌 Endpoints da API

A API expõe os seguintes endpoints na rota base `/fornecedores`:

* `POST /fornecedores`: Cria um novo fornecedor.
* `GET /fornecedores`: Lista todos os fornecedores cadastrados.
* `GET /fornecedores/{id}`: Busca um fornecedor específico pelo seu ID.
* `DELETE /fornecedores/{id}`: Deleta um fornecedor específico pelo seu ID.
